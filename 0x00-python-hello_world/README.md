# 0x00 Python - Hello, World

Welcome to the beginner's Python project on introductory programming concepts, focusing on basic scripts and the "Hello, World" program.

## Overview

This project is designed for beginners to:
- Execute Python scripts.
- Print text using Python.
- Understand basic Python syntax.

## Requirements

- Python 3.8 or above.
- Adherence to PEP 8 style guide.

## Files

| File Name        | Description                                      |
|------------------|--------------------------------------------------|
| `0-run`          | Shell script to run a Python script.             |
| `1-run_inline`   | Runs Python code from the command line.          |
| `2-print.py`     | Python script that prints a specific string.     |

## Tasks

### 0. Run Python file
- Write a Shell script that runs a Python script.

### 1. Run inline
- Execute Python code directly from the command line.

### 2. Hello, print
- Create a Python script that prints `"Hello, World"`.
